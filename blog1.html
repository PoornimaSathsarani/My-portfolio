Portfolio Website Project

October 2023 | Web Development

Overview

This project is a personal portfolio website designed to showcase my skills, experience, and projects in the field of Biomedical Technology and Web Development. It serves as a central hub for potential employers and collaborators to learn more about my work.


Technical Details

The website is built using pure HTML, CSS, and JavaScript without relying on heavy frameworks. This ensures fast load times and full control over the design. Key features include:

  • Fully responsive layout using CSS Grid and Flexbox.
  • Dark and Light mode toggle functionality.
  • Interactive elements like a typing effect and scroll animations.
  • Contact form integration.
