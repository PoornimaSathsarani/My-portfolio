University Practical Assessment

November 2023 | Academic Project

Overview

This web application was developed as part of a university practical assessment. The goal was to demonstrate a solid understanding of core web technologies while integrating biomedical concepts relevant to my degree.


Key Features

The project focuses on usability and accessibility within a healthcare context.

  • Interactive user interface for data entry.
  • Validation of biomedical data inputs.
  • Responsive design for tablet and mobile usage in clinical settings.
  • Clean and maintainable code structure.
